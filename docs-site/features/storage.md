# 网络存储

网络存储功能可以将服务器上的指定目录通过 WebDAV 或 SFTP 协议对外提供访问，方便远程管理文件。

## 功能概述

- 通过 WebDAV 协议提供文件访问（支持 Windows 资源管理器、macOS Finder 直接挂载）
- 通过 SFTP 协议提供文件访问（支持 FileZilla、WinSCP 等工具）
- 支持用户名/密码认证
- 支持只读或读写模式

---

## 配置说明

进入 **网络存储** 页面，点击 **新建** 按钮：

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| 名称 | 字符串 | ✅ | — | 存储名称 |
| 启用 | 布尔 | ✅ | `true` | 是否启用 |
| 协议 | 枚举 | ✅ | `WebDAV` | `WebDAV` 或 `SFTP` |
| 根目录 | 字符串 | ✅ | — | 对外暴露的本地目录路径 |
| 监听端口 | 整数 | ✅ | — | 服务监听端口 |
| 用户名 | 字符串 | ✅ | — | 访问用户名 |
| 密码 | 字符串 | ✅ | — | 访问密码 |
| 只读模式 | 布尔 | ❌ | `false` | 是否只允许读取，不允许写入 |

## 配置示例

### WebDAV 配置

| 字段 | 值 |
|------|-----|
| 名称 | 文件共享 |
| 协议 | WebDAV |
| 根目录 | `/home/user/files` |
| 监听端口 | `8081` |
| 用户名 | `admin` |
| 密码 | `your-password` |

### SFTP 配置

| 字段 | 值 |
|------|-----|
| 名称 | SFTP 访问 |
| 协议 | SFTP |
| 根目录 | `/home/user/files` |
| 监听端口 | `2222` |
| 用户名 | `sftp-user` |
| 密码 | `your-password` |

## 客户端连接方式

### WebDAV

**Windows 资源管理器：**
1. 打开"此电脑"，点击"映射网络驱动器"
2. 文件夹填写：`http://服务器IP:8081`
3. 输入用户名和密码

**macOS Finder：**
1. 菜单栏 → 前往 → 连接服务器
2. 服务器地址：`http://服务器IP:8081`
3. 输入用户名和密码

**命令行（Linux）：**
```bash
# 安装 davfs2
sudo apt install davfs2

# 挂载
sudo mount -t davfs http://服务器IP:8081 /mnt/webdav
```

### SFTP

**FileZilla：**
- 主机：`sftp://服务器IP`
- 端口：`2222`
- 用户名/密码：填写配置的凭证

**命令行：**
```bash
sftp -P 2222 sftp-user@服务器IP
```

## 注意事项

::: warning 安全建议
- 使用强密码，避免使用弱密码
- 建议只暴露必要的目录，不要暴露系统根目录
- 如果通过公网访问，建议配合 [访问控制](/features/access-control) 限制来源 IP
:::

::: tip HTTPS 加密
WebDAV 默认使用 HTTP 传输，数据未加密。如需加密传输，可以通过 [Caddy 反向代理](/features/caddy) 为 WebDAV 添加 HTTPS 支持。
:::
