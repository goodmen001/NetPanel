# 访问控制

访问控制功能支持配置 IP 黑白名单，可以只允许特定 IP 访问 NetPanel，或者屏蔽某些 IP。

## 功能概述

- **黑名单模式**：拒绝黑名单中的 IP 访问，其他 IP 正常访问
- **白名单模式**：只允许白名单中的 IP 访问，其他 IP 全部拒绝
- 支持单个 IP 和 CIDR 网段
- 支持 IPv4 和 IPv6

---

## 配置说明

进入 **访问控制** 页面：

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| 启用访问控制 | 布尔 | ✅ | `false` | 是否启用 IP 访问控制 |
| 控制模式 | 枚举 | ✅ | `黑名单` | `黑名单` 或 `白名单` |
| IP 列表 | 文本 | ✅ | — | 每行一个 IP 或 CIDR 网段 |

### IP 格式说明

| 格式 | 示例 | 说明 |
|------|------|------|
| 单个 IPv4 | `192.168.1.100` | 精确匹配单个 IP |
| IPv4 网段 | `192.168.1.0/24` | 匹配 192.168.1.0 ~ 192.168.1.255 |
| 单个 IPv6 | `2001:db8::1` | 精确匹配单个 IPv6 |
| IPv6 网段 | `2001:db8::/32` | 匹配 IPv6 网段 |

---

## 配置示例

### 示例 1：只允许局域网访问

白名单模式，只允许局域网 IP 访问 NetPanel：

| 字段 | 值 |
|------|-----|
| 控制模式 | 白名单 |
| IP 列表 | `192.168.0.0/16`<br>`10.0.0.0/8`<br>`172.16.0.0/12`<br>`127.0.0.1` |

### 示例 2：屏蔽特定 IP

黑名单模式，屏蔽已知的恶意 IP：

| 字段 | 值 |
|------|-----|
| 控制模式 | 黑名单 |
| IP 列表 | `1.2.3.4`<br>`5.6.7.0/24` |

---

## 注意事项

::: danger 白名单模式风险
启用白名单模式前，请确保你当前的 IP 已在白名单中，否则你将无法访问 NetPanel 管理界面。
:::

::: tip 与 EasyTier 配合
如果通过 EasyTier 异地组网访问 NetPanel，需要将 EasyTier 虚拟 IP 网段（如 `10.144.144.0/24`）加入白名单。
:::
