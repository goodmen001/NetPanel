# 计划任务

计划任务功能基于 Cron 表达式，支持定时执行 Shell 命令或发送 HTTP 请求，用于自动化运维操作。

## 功能概述

- 基于标准 Cron 表达式的定时调度
- 支持执行 Shell 命令
- 支持发送 HTTP 请求（GET/POST）
- 查看任务执行历史和日志

---

## Cron 表达式说明

Cron 表达式由 5 个字段组成（分 时 日 月 周）：

```
┌───────────── 分钟 (0-59)
│ ┌───────────── 小时 (0-23)
│ │ ┌───────────── 日 (1-31)
│ │ │ ┌───────────── 月 (1-12)
│ │ │ │ ┌───────────── 星期 (0-6，0=周日)
│ │ │ │ │
* * * * *
```

### 常用示例

| 表达式 | 说明 |
|--------|------|
| `* * * * *` | 每分钟执行一次 |
| `0 * * * *` | 每小时整点执行 |
| `0 2 * * *` | 每天凌晨 2 点执行 |
| `0 2 * * 0` | 每周日凌晨 2 点执行 |
| `0 2 1 * *` | 每月 1 日凌晨 2 点执行 |
| `*/5 * * * *` | 每 5 分钟执行一次 |
| `0 8-18 * * 1-5` | 工作日 8:00-18:00 每小时执行 |

## 配置说明

进入 **计划任务** 页面，点击 **新建** 按钮：

### 基础配置

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| 名称 | 字符串 | ✅ | — | 任务名称 |
| 启用 | 布尔 | ✅ | `true` | 是否启用 |
| Cron 表达式 | 字符串 | ✅ | — | 执行时间规则 |
| 任务类型 | 枚举 | ✅ | `Shell` | `Shell 命令` 或 `HTTP 请求` |

### Shell 命令配置

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 命令 | 字符串 | ✅ | 要执行的 Shell 命令 |
| 超时时间 | 整数 | ❌ | 命令执行超时（秒），0 表示不限制 |

### HTTP 请求配置

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| URL | 字符串 | ✅ | — | 请求地址 |
| 方法 | 枚举 | ❌ | `GET` | `GET` 或 `POST` |
| 请求头 | 文本 | ❌ | — | 自定义请求头，格式：`Key: Value` |
| 请求体 | 文本 | ❌ | — | POST 请求体内容 |
| 超时时间 | 整数 | ❌ | `30` | 请求超时（秒） |

## 配置示例

### 示例 1：定时备份数据库

每天凌晨 3 点备份 SQLite 数据库：

| 字段 | 值 |
|------|-----|
| 名称 | 每日数据库备份 |
| Cron 表达式 | `0 3 * * *` |
| 任务类型 | Shell 命令 |
| 命令 | `cp /var/lib/netpanel/netpanel.db /backup/netpanel-$(date +%Y%m%d).db` |

### 示例 2：定时清理日志

每周日凌晨 4 点清理 30 天前的日志：

| 字段 | 值 |
|------|-----|
| 名称 | 清理旧日志 |
| Cron 表达式 | `0 4 * * 0` |
| 任务类型 | Shell 命令 |
| 命令 | `find /var/log/netpanel -name "*.log" -mtime +30 -delete` |

### 示例 3：定时 HTTP 健康检查

每 5 分钟检查一次服务是否正常：

| 字段 | 值 |
|------|-----|
| 名称 | 服务健康检查 |
| Cron 表达式 | `*/5 * * * *` |
| 任务类型 | HTTP 请求 |
| URL | `http://127.0.0.1:3000/health` |
| 方法 | `GET` |

### 示例 4：定时发送 WebHook 通知

每天早上 8 点发送每日报告：

| 字段 | 值 |
|------|-----|
| 名称 | 每日报告 |
| Cron 表达式 | `0 8 * * *` |
| 任务类型 | HTTP 请求 |
| URL | `https://hooks.example.com/notify` |
| 方法 | `POST` |
| 请求体 | `{"text": "NetPanel 每日报告"}` |

## 查看执行历史

在任务列表中，点击任务右侧的 **历史** 按钮，可查看：
- 执行时间
- 执行结果（成功/失败）
- 命令输出或 HTTP 响应

---

## 注意事项

::: warning Shell 命令安全
Shell 命令以 NetPanel 进程的权限执行。请确保命令来源可信，避免执行危险操作（如 `rm -rf`）。
:::

::: tip 时区设置
Cron 表达式基于服务器本地时区。如果时区不正确，可以通过环境变量 `TZ=Asia/Shanghai` 设置时区。
:::
