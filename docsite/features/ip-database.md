# IP 地址库

IP 地址库功能提供 IP 归属地查询和批量管理能力，可以查询 IP 地址的地理位置信息。

## 功能概述

- 查询单个 IP 的归属地信息（国家、省份、城市、运营商）
- 批量导入和查询 IP 列表
- 支持导出查询结果

---

## 使用说明

### 单个 IP 查询

进入 **IP 地址库** 页面，在搜索框中输入 IP 地址，点击查询：

**查询结果包含：**
- 国家/地区
- 省份/州
- 城市
- 运营商/ISP
- IP 类型（家庭宽带、数据中心、代理等）

### 批量查询

点击 **批量查询** 按钮，在文本框中每行输入一个 IP 地址，支持一次查询多个 IP。

**支持格式：**
```
1.2.3.4
5.6.7.8
192.168.1.1
```

---

## 配置说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| IP 数据库 | 文件 | ❌ | 上传自定义 IP 数据库文件（MaxMind GeoIP2 格式） |
| 自动更新 | 布尔 | ❌ | 是否自动更新 IP 数据库 |

---

## 使用场景

- **访问日志分析**：查询访问者的地理位置
- **安全审计**：识别异常来源 IP
- **DDNS 调试**：验证 IP 归属地是否符合预期
- **访问控制**：结合 IP 归属地配置访问规则

---

## 注意事项

::: info 数据库精度
IP 归属地数据基于 IP 数据库，精度取决于数据库的质量和更新频率。对于家庭宽带 IP，精度通常到城市级别；对于数据中心 IP，精度更高。
:::

::: tip 自定义数据库
如果需要更精确的 IP 归属地数据，可以上传 MaxMind GeoIP2 格式的数据库文件。MaxMind 提供免费的 GeoLite2 数据库，可从其官网下载。
:::
